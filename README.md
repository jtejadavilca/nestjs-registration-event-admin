## Description

Project using NestJS+TypeORM+Postgres with teaching purposal. It is going to be used for managing of events and participants, and will be used only by administrator.

## Project setup

- Run DB using docker (configure .env file, coping it from `.env.template`)

```bash
$ docker compose up -d
```

- Install dependencies

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
